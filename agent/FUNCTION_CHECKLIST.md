# 我爱我家系统 - 功能清单

## 已完成功能 ✅

### 基础架构
- [x] 用户注册/登录 (JWT认证)
- [x] 邀请码系统（注册 + 点菜）
- [x] 用户权限管理（普通用户/管理员）
- [x] MySQL 数据库
- [x] Redis 缓存
- [x] Nginx 反向代理
- [x] 统一登录入口 + 系统选择页

### 点菜系统
- [x] 菜品录入（图片、名称、分类、口味、难度等）
- [x] 菜品浏览（网格视图、搜索、筛选）
- [x] 菜品详情（大图展示、编辑、删除）
- [x] 菜品分类管理（CRUD）
- [x] 口味标签管理
- [x] 点菜功能（选择人数、选择菜品）
- [x] 菜单历史（保存、查看、删除）
- [x] 邀请点菜（生成二维码、微信分享）
- [x] 华为手机微信分享适配

### 媒体管理系统
- [x] 图片上传 (JPEG, PNG, GIF, WebP)
- [x] 视频上传 (MP4, MOV, AVI, MKV, WebM)
- [x] 批量上传 (最多50个文件)
- [x] EXIF 元数据提取（拍摄时间、GPS、相机信息）
- [x] 自动生成缩略图（图片 + 视频）
- [x] 网格/列表视图切换
- [x] 搜索功能（标题、描述）
- [x] 分页功能
- [x] 文件类型筛选（图片/视频）
- [x] 时间范围筛选
- [x] 标签系统（CRUD、自定义颜色）
- [x] 分类系统（CRUD、层级支持）
- [x] 媒体详情（大图预览、信息展示）
- [x] 图片旋转
- [x] Canvas 图片编辑器
- [x] 用户数据隔离（标签、分类按用户隔离）

### 心愿系统
- [x] 系统需求提交
- [x] 心愿墙（卡片式布局）
- [x] 状态跟踪（接收/开发中/完成）
- [x] 管理员回复功能
- [x] 创建者编辑/删除

### 管理员功能
- [x] 邀请码管理（创建、查看、删除）
- [x] 支持自动生成和自定义邀请码
- [x] 邀请码过期时间设置
- [x] 区分注册邀请码和点菜邀请码

### 性能优化
- [x] Redis 缓存（媒体列表、详情）
- [x] 图片懒加载
- [x] 静态文件缓存
- [x] 批量操作
